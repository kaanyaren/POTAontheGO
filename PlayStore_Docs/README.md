# PlayStore Upload Documentation

This directory contains all the documentation needed to prepare and submit the POTAontheGO app to the Google Play Store. Follow these guides step-by-step to ensure a successful release.

- `overview.md` – high-level requirements and process
- `checklist.md` – pre-submission checklist items
- `metadata.md` – Play Store listing metadata template
- `icons_and_screenshots.md` – asset specifications and naming guidelines
