# POTA on the Go

**POTAontheGO** is a mobile application built with Flutter that helps amateur radio operators participate in the Parks on the Air (POTA) program. The app provides maps, park details, and other tools to locate and log contacts from national and state parks.

## Repository Structure

- `pota_on_the_go/` – Flutter project containing the main application
  - `lib/` – Dart source code
  - `android/`, `ios/`, `web/`, `windows/`, `macos/`, `linux/` – platform-specific build files
  - `test/` – unit and widget tests
- `PlayStore_Docs/` – documentation for preparing a Play Store release (checklists, metadata, icons)
- `docs/` – static website or additional documentation assets
- `assets/` – shared assets such as icons and fonts used across modules

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- Android/iOS development environment (Android Studio, Xcode, etc.)

### Build & Run

From the workspace root:

```bash
cd pota_on_the_go
flutter pub get
flutter run   # choose a connected device or emulator
```

For platform-specific builds:

```bash
flutter build apk      # Android
flutter build ios      # iOS
flutter build web      # Web
# see flutter build documentation for other targets
```

### Testing

Run the test suite with:

```bash
cd pota_on_the_go
flutter test
```
